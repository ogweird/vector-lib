## vector-lib  
Simple C++ vector library usable in game development and math  
Made for C++ 17 and newer  

## Copyright  
The library is free to use with no need to credit me as author  

## Usage and examples  
```C++
#include "vector.hpp"

Vector2 test_vector_2d = { 4, 5 };
Vector3 test_vector_3d = { 1, 3, 9 };
```
